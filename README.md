# Nexus Hub 🌐

Nexus Hub is a sleek, modern web-based utility that provides a detailed dashboard of your system, browser, and network information. It's designed with a clean, responsive, and user-friendly interface, offering comprehensive insights at a glance.

## ✨ Features

Nexus Hub presents a wide range of information organized into intuitive cards:

#### **Comprehensive Information Panels**
* **📍 Network Details:** Fetches IPv4/v6 addresses, Geolocation (Country, City, Region), ISP, ASN, and network timezone.
* **💻 System Information:** Detects your Operating System, Device Type, CPU Cores, estimated Memory, and Architecture.
* **🌐 Browser Details:** Identifies the browser name, version, rendering engine, and full User Agent string.
* **🎨 Display & Graphics:** Shows screen & viewport resolution, GPU renderer, pixel ratio, and color depth.
* **🛡️ Security & Privacy:** Checks for HTTPS status, "Do Not Track" signals, WebRTC support, and Local/Session Storage availability.
* **⚡ Web Features:** Confirms support for JavaScript, WebGL, Geolocation API, and Touch events, plus displays your real-time online status.

#### **Interactive UX/UI**
* **🔍 IP Lookup:** An integrated search bar to look up details for any valid IP address.
* **📋 Copy to Clipboard:** Easily copy key information like your IP address, User Agent, or ASN with a single click.
* **🌗 Dark/Light Theme:** Seamlessly switch between dark and light modes. The app also respects and defaults to your system's preferred theme.
* **🌍 Multi-language Support:** Full i18n support with translations for 10+ languages, including English, Spanish, German, Persian, and more.
* **📱 Fully Responsive:** A fluid layout that adapts beautifully to any screen size, from large desktops to small mobile phones.
* **💎 Modern Design:** A polished interface featuring smooth loading animations, icon-driven cards, and a frosted glass (glassmorphism) effect.

## 🛠️ Technologies Used

This project is built with a focus on performance and simplicity, using only vanilla web technologies.

* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
* **APIs:** [IPify](https://www.ipify.org/) & [FindIP](https://findip.net/) for network and geolocation data.
* **Fonts & Icons:**
    * [Google Fonts (Inter)](https://fonts.google.com/specimen/Inter)
    * [Phosphor Icons](https://phosphoricons.com/)
    * [Circle Flags](https://github.com/hatscripts/circle-flags) for language selection.

## 🚀 Getting Started

No complex build steps or dependencies are required. Simply clone the repository and open the `index.html` file in your browser.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/WhoisNeon/Nexus-Hub.git](https://github.com/WhoisNeon/Nexus-Hub.git)
    ```
2.  **Navigate to the directory:**
    ```bash
    cd Nexus-Hub
    ```
3.  **Open the file:**
    Open `index.html` in your favorite web browser.
