# System Info Hub 🌐💻
**Get detailed information about your system, browser, network, and more, all in one place.**

[![Live Demo](https://img.shields.io/badge/Demo-Live%20Demo-green?style=for-the-badge)](https://whoisneon.github.io/System-Info-Hub)

System Info Hub is a sleek and modern web application designed to quickly gather and display a comprehensive overview of your device's specifications, browser details, and network information. Built with a focus on user experience and clarity, it's perfect for developers, tech enthusiasts, or anyone curious about their current environment.

## ✨ Features

* **Network Information:**
    * Public IP Address
    * Public IPv6 Address
    * Geographic Location (City, Country)
    * Internet Service Provider (ISP)
    * Connection Type (e.g., AS-number)
    * IP Time Zone
* **Browser Details:**
    * Browser Name (Chrome, Firefox, Safari, Edge, Opera, IE)
    * Browser Version
    * Rendering Engine (Blink, Gecko, WebKit, Trident)
    * Full User Agent String
    * Cookies Enabled Status
* **System Information:**
    * Operating System (Windows, macOS, Android, iOS, Chrome OS, Linux)
    * Device Type (Desktop, Mobile, Tablet)
    * System Architecture (e.g., Win32)
    * CPU Cores
    * Device Memory (in GB)
    * System Time Zone
* **Display & Graphics:**
    * Screen Resolution
    * Viewport Size
    * Color Depth
    * Graphics Processing Unit (GPU) detection
    * Device Pixel Ratio
* **Security & Privacy:**
    * HTTPS Status (Secure/Insecure)
    * "Do Not Track" (DNT) Status
    * WebRTC Support
    * Local Storage Availability
    * Session Storage Availability
* **Web Features:**
    * JavaScript Enabled Status
    * WebGL Support
    * Geolocation API Support
    * Touch Support
    * Online/Offline Status
* **User-Friendly Interface:**
    * Responsive design for various screen sizes
    * Dynamic theme toggling (Light/Dark mode) with system preference detection
    * Live updates for viewport size and online status
    * Copy-to-clipboard functionality for IP Address

## 🚀 Technologies Used

* **HTML5:** Structure of the web application
* **CSS3:** Styling and responsive design
* **JavaScript (ES6+):** Core logic and dynamic content updates
* **Phosphor Icons:** For clear and aesthetic iconography
* **Inter Font:** Modern and readable typography
* **Theme Toggles Library:** For the beautiful dark/light mode switch
